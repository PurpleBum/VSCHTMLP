Come Code with Kolade

Code With K or Code with Kolade is a site created to help showcase the work of Kolade its programmer and establish an online presence showing young African Americans that coding is simple.

Code With K wants to show that the process of learning HTML, CSS, and other programming languages aren't that difficult once you have an idea of were to begin!
